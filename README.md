# 🌐 My Blog Website

A simple and responsive **Blog Website** built using HTML, CSS, and JavaScript. This project demonstrates how to create a clean UI, manage multiple pages dynamically, and implement basic interactivity without using any frameworks.

---

## 🚀 Live Demo

👉 https://vasanth-web-dev.github.io/Blog-Site/

---

## 📘 Description

This is a single-page blog application that simulates multiple pages (Home, Blogs, and Post) using JavaScript. It includes features like dynamic content loading, blog search functionality, and smooth navigation—all within one HTML file.

---

## ✨ Features

- 🏠 Home page with hero section and featured posts  
- 📚 Blog listing page with search functionality  
- 📝 Dynamic blog post rendering  
- 🔍 Real-time search filter  
- 🔄 Page navigation without reload  
- 🎨 Clean and modern UI design  
- 📱 Fully responsive layout  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox + Responsive Design)  
- JavaScript (DOM Manipulation)  
- Google Fonts (Poppins)  

---

## ⚙️ How It Works

- All pages (Home, Blogs, Post) are inside a single HTML file  
- JavaScript controls page visibility using `.active` class  
- Blog content is stored in a JavaScript object  
- Clicking "Read More" dynamically loads content into the post section  
- Search filters blog cards based on user input
